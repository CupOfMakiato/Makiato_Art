# Makiato Art Website

A simple website for sharing commission information and Linktree-style social links.

## What this site includes

- Commission details and process information
- Contact section for commission inquiries
- FAQ and Terms of Service pages
- Linktree-style quick links to social platforms

## Tech Stack

- React + Vite
- React Router
- Tailwind CSS
- ESLint

## Local Development

```bash
npm install
npm run dev
```

App runs on the local Vite dev server (usually `http://localhost:5173`).

## Scripts

- `npm run dev` - start development server
- `npm run build` - create production build
- `npm run preview` - preview production build locally
- `npm run lint` - run lint checks

## Deployment

This project is configured for deployment with Vercel and includes GitHub Actions workflows for CI/CD.
